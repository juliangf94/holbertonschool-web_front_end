# Web Front End - HTML Advanced

## 📝 Description
This project is part of the **Holberton School** curriculum for the Web Front End specialization. The objective is to build a professional-grade digital agency website called **Techium** and a comprehensive **Styleguide**. 

Through these tasks, I have mastered advanced HTML techniques, semantic structure, accessibility (ARIA), and multimedia integration without relying on CSS for styling yet.

## 🚀 Learning Objectives
* Use semantic HTML5 elements correctly (`<header>`, `<main>`, `<footer>`, `<section>`, `<article>`).
* Implement advanced multimedia elements (`<video>`, `<audio>`, `<iframe>`).
* Create interactive components using only HTML (`<details>`, `<summary>`).
* Optimize accessibility through the use of `alt` attributes, `titles`, and proper heading hierarchy.
* Organize complex document structures with comments and clear sectioning.
